# Neo Slack Bot - User Guide

## Overview

Neo Slack Bot is a native macOS Slack bot client that provides real-time Slack event processing and macOS notification integration. This guide explains how to use the bot, configure it, and verify its operation.

## Documentation Structure

The user guide is organized into the following documents:

- **[User Guide - Quick Start](user-guide-quick-start.md)** - Quick start, prerequisites, basic usage, and verifying bot operation

- **[User Guide - Testing](user-guide-testing.md)** - Testing message sending/receiving, multiple users, channels, and notifications

- **[User Guide - Configuration](user-guide-configuration.md)** - Configuration file, environment variables, manual testing checklist, CLI testing commands, and troubleshooting

- **[User Guide - Advanced Usage](user-guide-advanced.md)** - Running multiple bot instances, monitoring bot activity, performance monitoring, best practices, and getting help
