# Features Detailed

## Overview

This document provides an overview of detailed technical documentation for Neo Slack Bot features. The detailed information is split across multiple documents for better organization.

## Documentation Structure

The detailed features documentation is organized into the following documents:

- **[Features - Core Implementation](features-core.md)** - Core features implementation details, including Socket Mode, Event Processing, Notifications, HTTP API, Message Filtering, Command Handling, User Preferences, Logging, Error Recovery, and Rate Limiting

- **[Features - Technical Details](features-technical.md)** - Technical implementation details, including C++17 features, Async Event Processing, Thread Safety, Resource Management, and Clean Architecture

- **[Features - Platform Details](features-platform.md)** - Platform-specific features, including macOS Native Integration, Background Operation, and Low Resource Usage

## Quick Reference

For a quick overview of all features, see [Features Overview](features-overview.md).
