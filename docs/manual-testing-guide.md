# Manual Testing Guide for Neo Slack Bot

## Overview

This guide provides step-by-step instructions for manually testing Neo Slack Bot functionality. The guide is organized into multiple documents for better navigation.

## Documentation Structure

The manual testing guide is organized into the following documents:

- **[Manual Testing Guide - Setup](manual-testing-guide-setup.md)** - Prerequisites, test environment setup, and quick start

- **[Manual Testing Guide - Basic Test Scenarios](manual-testing-guide-scenarios-basic.md)** - Basic test scenarios including message receiving, commands, multiple users, and mentions

- **[Manual Testing Guide - Advanced Test Scenarios](manual-testing-guide-scenarios-advanced.md)** - Advanced test scenarios including filtering, notifications, edge cases, and multiple bot instances

- **[Manual Testing Guide - Advanced](manual-testing-guide-advanced.md)** - CLI testing commands, verification checklist, troubleshooting, advanced testing, and test results template
